Amazon Lamdba で登録されている関数一覧を取得する方法について調査した内容を備忘録として残しておく

## 方法

該当の AWS Console にログインして AWS Cloud Shell 上で以下のコマンドを実行する

## 実際のコード

```bash
aws lambda list-functions --region {リージョン名} \
| jq '.Functions[] | [.FunctionName]' \
| jq -rs 'map([.[]])[] | @csv' \
| tr -d '"'
> lambda-functions-list.csv
```
