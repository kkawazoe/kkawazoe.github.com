draw.io MCP を各ツールに導入する手順を備忘録として残しておく

## 環境

- OS: MacOS Tahoe 26.3

### Cursor

#### 前提

- Node.js がインストールされていること

1. Cursor Settings を開く
2. サイドバー > 「Tools & MCP」 を選択して、「Add Custom MCP」を選択

    {{<
      img
        src="/images/20260305_01/cursor_config_1.png"
        href="/images/20260305_01/cursor_config_1.png"
        caption="Cursor Settings > Tools & MCP 画面"
        class="center"
        w="900"
        h="400"
    >}}

3. 以下の内容を入力して保存する

    ```json:mcp_servers.json
    {
      "mcpServers": {
        "drawio": {
          "command": "npx",
          "args": [
            "-y",
            "@drawio/mcp@latest"
          ]
        }
      }
    }
    ```

4. Installed MCP Servers に draw.io MCP が追加されている & 有効になっていることを確認する
