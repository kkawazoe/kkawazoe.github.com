## Gretel について

パンくずリストを表示する Gem  
[github](https://github.com/WilHall/gretel)

## 実際のコード

[config/breadcrumbs.rb]
```ruby

# articles#index
crumb :articles do |article|
  icon = article.flg ? image_tag("表示したい画像") : ""
  link icon + {表示文字}, {URL}
  parent :parent_articles, article.category
end
```
